# プログラム担当のための環境構築のやり方

この文章は**プログラムを書く担当の人向け**です  
プログラムを直接書かない人はこの手順はやらなくていいです

## IDE のインストール

まず、**[このサイト](https://www.jetbrains.com/ja-jp/idea/download)** から **Intellij IDEA** というソフトをダウンロードします

![ダウンロード画面](https://i.gyazo.com/9ad42de520b1f17224d84aa6cda45f24.png)

二つ選択肢がありますが、今回は**右側の Community** をダウンロードしてください

ダウンロードが終了したら、パソコンの **ダウンロード** というフォルダを開いてください

![ダウンロードのフォルダ画面](https://i.gyazo.com/b3d829ca37cb988dc7296396b6b8991d.png)

ここにある **idealC うんたら.exe** をダブルクリックして、そのあとに出てくるウィンドウにしたがってインストールを完了させてください

> 今回、このチームでは kotlin というプログラミング言語を用いて開発を行います  
> この **Intellij IDEA** というソフトは kotlin を使って、快適にプログラミングを行うためのソフトウェアです  
> このようなソフトウェアを **IDE(統合開発環境)** と呼びます

## Git のインストールとアカウント作成

### ダウンロード

次に **Git** というソフトをダウンロードしましょう

**[このサイト](https://gitforwindows.org/)** の中央にある Download を押してダウンロードしてください

ダウンロードが完了したら Intellij IDEA と同じように、ダウンロードフォルダからインストーラを実行してください

ここから先のダウンロードの手順については、**[このサイト](https://sukkiri.jp/technologies/devtools/git/git_win.html)** が非常にわかりやすく解説しているので参考にしてください

### 初期設定

Git を使うためには Git アカウントを作成する必要があります

初期設定のやり方についても **[例のサイト](https://sukkiri.jp/technologies/devtools/git/git_basic_use.html)** が非常に分かりやすいので(以下略)

登録する際は  
**ユーザー名: 漢字フルネーム  
メールアドレス: 学校の Email アドレス**  
で今回は登録してください

ここまでできたら念のため、一度 PC を再起動しておきましょう

> Git とはプログラムのコードのバージョンを管理するためのソフトウェアです  
> **誰が、いつ、どのような変更をプログラムに加えたのか**を管理するために使用します

## Github のアカウント作成

次に **[github.com](https://github.com/)** にアクセスして、右上の **Sign up** を押してください

### メールアドレス入力

![githubのemail登録画面](https://i.gyazo.com/60fff58b44a56fb213beec46abdbfab7.png)

上の画像のように表示されたら**学校のメールアドレス**を入力してください  
入力が終わったら **Continue** を押して次に進みましょう

### パスワード入力

次に、パスワードを設定しましょう  
この時、**Password is strong** と下に表示されるような、強固なパスワードを設定することをオススメします (**パスワードを忘れないでください**)

### ユーザーネーム

ユーザーネームを設定します  
これはかぶらない ID が必要なので、
今回は **`下の名前_kanakou`** で登録しましょう  
`例) jinnosuke_kanakou`  
(どうしても嫌なら他のでもいいです)

### メールアドレス受信設定

```bash
Would you like to receive product updates and announcements via email?
Type "y" for yes or "n" for no

→
```

上のように出てくるので `n` と入力してください (メールいらないよね？)

### ロボットか確認

ロボットではないことを確認するやつが出てくるのでロボットじゃないことを証明してください

### メールアドレスの確認

メールに送られたコードを入力して Verify してください

### なんか押すやつ

Student 押して continue しちゃってください

### プランの選択

なんと Github は学生だとタダで有料プランくれます！  
でもめんどいんで今回は Free でいいです...(左のほう)

> Github は Git のリモートリポジトリをホスティングするサービスです
>
> 簡単に言うと、みんなで開発するにあたって、今最新のバージョンはどれなんだ！？とならないようにするためのサービスです
>
> Git はローカル(PC の中)を管理します  
> Github はローカルの変更をまとめておいておく場所です
>
> 使いこなすと仕事にも生かせるかもしれないので、ぜひ自分で勉強してみてください (調べると情報がいっぱい出てきますし、本もたくさんあります)

## Github と Git の連携

ここまでで、Git と Github それぞれのセットアップが完了しました  
しかし、**まだ Git と Github は結びついていません**  
なので、ここからは連携作業をしていきましょう

連携には https を用いる方法と、ssh を用いる方法があります  
今回は安全性が高い ssh での連携をしましょう

### 秘密鍵と公開鍵を生成する

なんか難しい単語ですが、深く考えないでください  
要するに、家の鍵とスペアキーを作るようなものです

先ほどインストールした **Git(Git Bash)** を使います  
デスクトップで**右クリック**すると  
![右クリメニュー](https://i.gyazo.com/82be28e488d068f922aa17fbc3fdfb16.png)  
こんなかんじに`Git GUI Here`と`Git Bash Here`という二つのメニューが増えていると思います  
ですので、`Git Bash Here`をクリックしてください  
![ハッカーぽい画面](https://i.gyazo.com/972eab00df2c757a687976dea7dd0311.png)
このハッカーっぽい画面に次のように入力してください

```bash
ssh-keygen -t ed25519 -C "学校のメールアドレス"
```

すると次のように表示されます

```bash
Enter file in which to save the key (/Users/you/.ssh/id_rsa): [Press enter]
```

気にせず Enter を押してください

```bash
Enter passphrase (empty for no passphrase): [Type a passphrase]
```

とでてきたら、パスワードを入力してください **(忘れないで!!!)**  
また、この時**パスワードを入力しても表示されません**が、それはセキュリティ対策なので安心して入力してください

```bash
Enter same passphrase again: [Type passphrase again]
```

と出るので、もう一度入力してください

そうしたら、PC フォルダを開いて、`C:\ユーザー\自分のアカウント\.ssh`を開いてください  
その中にある`id_rsa.pub`というファイルをメモ帳で開いて、ctrl+a ですべてコピーしてください  
![ファイルの図](https://i.gyazo.com/84e78763fbe4e363a7165bc0f7c89d42.png)  
**(id_rsa と間違えないで)**

### Github へコピー

すべてコピーができたら、[Github](https://github.com/)へログインしてください

右上のアイコンを押して出たメニューの中から、Settings をクリックしてください  
![githubメニュー](https://i.gyazo.com/9cf7edf19602293388e6959adb658c13.png)

たくさんある項目の中から SSH and GPG keys という項目を選んでください  
![いっぱいある](https://i.gyazo.com/33ace434983a6ebaa56888aeb497f2cf.png)

そうしたら、右上の緑色 New SSH key を押してください

そして出てきた画面  
![でてきた画面](https://i.gyazo.com/080a43a251bb4d7141d6123002e106cc.png)  
の Key に先ほどコピーしたものを貼り付けてください  
title には first とでも入れておきましょう

Add SSH key を押して次へ進んでください

最後にもう一度 Git bash を開いて、

```bash
ssh -T git@github.com
```

と入力してください  
`(yes/no)`とか聞かれたら yes を入力してください

```bash
You've successfully authenticated うんたら
```

こんな感じになったら成功です
